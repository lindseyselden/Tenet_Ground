# Typography

The same tenets that shaped the generated world select the type: **neutral by mandate, idiosyncratic
only in the logotype** (TBD — see [`logo.md`](logo.md)). The type system is Rule #04 (Typographic
Neutrality) made concrete — see [`../governance-rules.md`](../governance-rules.md).

## The principles, as type

- **Recessive (Rule #04).** The type recedes — a neutral set, the value read before the typeface.
  The rule and its pass/fail test live in [`../governance-rules.md`](../governance-rules.md); this is
  it applied.
- **Architecture dominates the figure.** The typefaces play at scale. The asserted value is
  monumental; the audit detail is precise beneath it. The system over the individual, made
  typographic.
- **Clinical but human.** Three voices: precision and personality.
- **One system, one intent.** A uniform system with a single signature.

## The three faces

| Face | Role |
| --- | --- |
| **IBM Plex Sans** | Drives the intelligent infrastructure — display and interface. |
| **IBM Plex Mono** | The data voice — telemetry, audit detail, numeric registers. |
| **Newsreader** | Holds the human — editorial, longform, the sensate register. |

## Reference settings

- *One system. One intent.* — Plex Sans
- *THE INTENT IS VERIFIED.* — Plex Mono, uppercase
- *The numbers have nerves.* — Newsreader
